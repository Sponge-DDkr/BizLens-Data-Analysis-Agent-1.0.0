# 竞品动态报告 — 2024 年 Q3

> 情报周期：2024-07 至 2024-09 | 更新频率：月度 | 密级：内部

---

## 一、总体态势

Q3 是 SaaS 行业的传统旺季（企业年末采购预算启动），但今年华南市场的竞争激烈程度超出预期。至少 3 家竞品在 Q3 推出了力度空前的促销活动，其中**竞品 A 的"新客首年 1 折"**对基础版市场的冲击最为严重。

---

## 二、竞品逐个分析

### 2.1 竞品 A — 重点监控 ⚠️

| 维度 | 详情 |
|------|------|
| 融资状态 | 2024-05 完成 A 轮 8000 万元 |
| Q3 大动作 | 8 月中旬推出"新客首年 1 折"，覆盖基础版全线 |
| 目标市场 | 华南 10-50 人小微企业（与我们基础版 100% 重叠） |
| 投放渠道 | 深圳/广州地铁广告 + 抖音信息流 + 企微社群裂变 |
| 预估获客成本 | 新客 CAC 约 800 元（行业平均 1200 元），明显在烧钱抢市场 |
| 预计持续时间 | 至少至 2024 年底（融资足够支撑 18 个月） |

**我们的应对**：不能跟进价格战（基础版毛利率本就偏低），必须走差异化路线。竞品 A 目前**不支持 API 对接外部系统**，这是我们的核心差异化卖点，需要在华南市场强化传播。

### 2.2 竞品 B — 本地化威胁

- 6 月完成 5000 万 A 轮融资
- 基础版年费从 9800→5800（-41%）
- 粤语客服 + 本地部署，在东莞/佛山侵蚀我们的客户
- **致命短板**：不支持 API 集成，无法接入企业现有系统

### 2.3 竞品 C — 全国性厂商

- 产品成熟但定价高（基础版 2.5 万/年），在华南中小企业市场无竞争力
- 2024 Q3 在华南无特殊动作

---

## 三、行业风向

1. **低价潮蔓延**：至少 5 家 SaaS 厂商在 Q3 降价或推出免费版，行业正在经历一轮"价格洗牌"
2. **AI 功能成为标配**：头部竞品 9 月陆续上线 AI 辅助功能（智能报表、AI 客服），我们需加速跟进
3. **本地化服务溢价消失**：过去"本地团队、本地服务"可溢价 20%，现在客户认为这是基本配置

---

## 四、风险评估矩阵

| 风险 | 概率 | 影响 | 应对 |
|------|------|------|------|
| 竞品 A 价格战持续侵蚀基础版份额 | 高 | 高 | 走差异化，强化 API 集成优势 |
| 华南新客获客成本进一步上升 | 高 | 中 | 加大老客转介绍激励，降低对投放依赖 |
| 竞品推出 API 集成功能（补上短板） | 中 | 高 | 提前构筑生态壁垒（集成数量>50 个） |
| 竞品推出 AI 功能差异化 | 中 | 中 | Q4 优先上线智能报表功能 |
